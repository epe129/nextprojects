# React + Vite ja flask

# Kirjahylly

Tee sovellus, jossa käyttäjät voivat lisätä ja hallita kirjojensa kokoelmaa. 
Flask tallentaa kirjat ja niiden tiedot (esim. nimi, tekijä, kuvaus) tietokantaan, 
ja React näyttää ne kauniisti listana.

Reactin osalta: Luo lomake kirjan lisäämistä varten ja listaa kirjat.

Flaskin osalta: Käytä Flaskia tallentamaan kirjat ja niiden tiedot SQL-tietokantaan.
